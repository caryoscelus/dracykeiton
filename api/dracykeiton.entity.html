dracykeiton.entity.declarative.properties(**props)[source]¶

Decorator on Entity class defining dynamic properties

Property default values can either be persistent values (numberes, strings, boolean) or functions creating complex values. DO NOT pass complex (i.e. changable) objects there, that will result in every instance having same object!

dracykeiton.entity.entity.simplenode(f)[source]¶

Returns function, which returns SimpleNode

>>> from dracykeiton.entity import simplenode, SimpleNode
>>> node = simplenode(int)(base=2)
>>> isinstance(node, SimpleNode)
True

>>> @simplenode
... def foo(a, b, value):
...     return (a+b)*value
>>> foo(1, 2).process(5)
15
