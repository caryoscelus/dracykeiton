dracykeiton.util.curry.curry(f)[source]¶

Takes a function and returns callable which returns callable.

>>> cint = curry(int)
>>> cint(base=2)('101')
5
>>> pprint = curry(print)('*')
>>> pprint(1)
* 1
>>> pprint.__name__
'print'
